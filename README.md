
# CodeKata Python Practice

This repository contains Python solutions to coding problems from [CodeKata](https://www.codekata.com/), organized using Jupyter Notebooks.

---

## Repository Structure

- Problems are grouped by difficulty level:
  - `easy/`
  - `medium/`
  - `hard/`

- Each folder contains Jupyter Notebooks (`.ipynb`) with up to 20 problems per notebook.
- Each notebook includes:
  - Problem statement (copied or summarized)
  - Python solution
